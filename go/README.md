# Berlin hackathon sample for go
This is golang server 

## Requirement
* golang

## Installation
(in GOPATH)
```bash
$ git clone https://github.com/couger-inc/berlin_hackathon.git
$ cd berlin_hackathon/go
$ go get -u
```

## Usage
```bash
$ go run *.go
```
