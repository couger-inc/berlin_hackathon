# Berlin hackathon sample
This is node js server 

## Requirement
* nodejs

## Installation

```bash
$ git clone https://github.com/couger-inc/berlin_hackathon.git
$ cd berlin_hackathon
$ npm i
```

## Usage
```bash
$ npm start
```
